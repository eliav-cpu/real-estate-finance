# 02 - User Guide

## Goal

Use this engine to test a custom payment schedule for the shell component only.

## Inputs

- base shell price
- proposal date
- delivery date
- annual rate
- payment dates
- payment percentages

## Excluded Items

Finish package, furniture, parking and service fees are calculated outside this engine.

## Output

The engine returns the adjusted shell price, economic gap and approval status.

## Rule

Sales can prepare a proposal. Finance or management approves the final custom plan.
